# Kanban Board Pro

Aplicação web de gerenciamento de tarefas inspirada em quadros Kanban, desenvolvida com HTML, CSS e JavaScript.

## Funcionalidades

- Adição de tarefas por coluna
- Drag and drop entre colunas
- Edição de tarefas
- Remoção individual
- Limpeza completa do quadro
- Contadores por coluna
- Persistência de dados com localStorage
- Interface responsiva

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo do projeto

Praticar manipulação do DOM, eventos, drag and drop, armazenamento local e construção de interfaces interativas para aplicações web.

## Como executar

1. Baixe ou clone este repositório
2. Abra o arquivo `index.html` no navegador
